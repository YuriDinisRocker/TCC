  # CellVision AI — Resumo do TCC

**Orientador:** Ademir Luiz do Prado
**Curso:** TADS (Análise e Desenvolvimento de Sistemas)

## Visão Geral

Sistema de classificação de imagens celulares em tempo real, voltado para auxiliar na análise de células sanguíneas a partir de imagens de microscopia.

## Stack Tecnológica

- **IA/Visão Computacional:** Python, TensorFlow/Keras, MobileNetV2 (transfer learning), OpenCV
- **Rastreamento:** Tracker de células baseado em SORT simplificado
- **Backend de IA:** FastAPI com WebSocket, transmissão de frames em Base64 JPEG renderizados em canvas HTML
- **Backend:** .NET
- **Dataset:** BCCD (Blood Cell Count and Detection)
